CHAT WIDGET DEBUG INSTRUCTIONS
================================

The chat widget has been updated with comprehensive debugging. Follow these steps:

STEP 1: Clear Browser Cache
----------------------------
1. Press Ctrl + Shift + Delete
2. Select "Cached images and files"
3. Click "Clear data"
4. Close browser completely

STEP 2: Open Website Fresh
---------------------------
1. Open your browser
2. Navigate to your website (open index.html)
3. Press F12 to open Developer Console
4. Go to "Console" tab

STEP 3: Check Console Messages
-------------------------------
You should see these messages when page loads:

✓ "Chat elements found: {chatToggle: true, chatPanel: true, ...}"
✓ "Attaching click event to chatToggle"

If you see "chatToggle: false" or errors, the HTML elements aren't loading.

STEP 4: Click Chat Button
--------------------------
1. Click the gradient circle button (bottom-left)
2. Watch the console for:
   ✓ "Chat toggle clicked!"
   ✓ "Opening chat..."

3. The chat panel should slide up from the bottom

STEP 5: Test Chat Functionality
--------------------------------
1. Type a message in the input
2. Click Send or press Enter
3. You should see:
   - Your message appear (blue background, right side)
   - "Typing..." indicator
   - AI response from Gemini API

COMMON ISSUES & FIXES:
======================

Issue 1: "chatToggle: false" in console
Fix: Check if index.html has <button id="chatToggle">
     The ID must be exactly "chatToggle" (case-sensitive)

Issue 2: Chat button visible but click does nothing
Fix: Check console for JavaScript errors
     Make sure script.js is loaded (check Network tab in F12)

Issue 3: Chat opens but no response
Fix: Check Gemini API key is correct in script.js line 408
     Check internet connection
     Look for "Gemini API Error" in console

Issue 4: Icons not showing
Fix: Hard refresh (Ctrl + Shift + R)
     Check Font Awesome CDN is loading (Network tab)

MANUAL TEST:
============
Open browser console and type:

document.getElementById('chatToggle').click()

If chat opens, the button works but CSS might be hiding it.
If nothing happens, check for JavaScript errors.

VERIFY CSS:
===========
In console, type:

getComputedStyle(document.getElementById('chatPanel')).display

Should show: "block" or similar (not "none")

When chat is open, type:

document.getElementById('chatPanel').classList.contains('open')

Should return: true

LAST RESORT:
============
If nothing works, check:
1. Are there multiple script.js files?
2. Is there a JavaScript error preventing code from running?
3. Try opening in Incognito/Private mode
4. Try a different browser
