TROUBLESHOOTING:
----------------

If chat doesn't open:
1. Hard refresh: Ctrl + Shift + R
2. Clear cache: Ctrl + Shift + Delete
3. Check browser console (F12) for errors

If AI doesn't respond:
1. Check internet connection
2. Verify API key is correct
3. Check console for API errors

CUSTOMIZATION:
--------------

To change chat position:
- Edit .chat-toggle and .chat-panel in style.css
- Change bottom/left values

To change colors:
- Edit CSS variables in style.css
- --primary, --violet, --accent

To change AI behavior:
- Edit the prompt in sendToGemini() function
- Modify temperature, maxOutputTokens in script.js
